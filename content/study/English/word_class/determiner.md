# Why Determiner ⊂ Adjective
## Reason: 'adj + determiner + noun' is impossible
- `adj` + `noun` &Rightarrow; (O)
- `adj` + `adj` + `noun` &Rightarrow; (O)
- `det` + `adj` + `noun` &Rightarrow; (O)
- `adj` + `det` + `noun` &Rightarrow; (X)

# Classification
## Atricle
- Indefinite Article: `a` / `an`
- Definite Article: `the`
